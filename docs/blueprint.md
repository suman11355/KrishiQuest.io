# **App Name**: KrishiQuest

## Core Features:

- Interactive Digital Twin Farm: Visual representation of the user's farm reflecting the selected crop during profile setup.
- Personalized Quest Recommendations: AI-powered tool that recommends quests based on user's crop, location, and current level.
- Gamified Profile Setup: Interactive form for setting up user profile and farm details with avatar selection and crop choices.
- Community Leaderboard: Leaderboard displaying top farmers in the user's Panchayat and their scores, fostering competition and engagement.
- Kisan Bazaar Rewards Redemption: Marketplace for redeeming Kisan Koins for discounts, virtual goods, and expert advice.
- Progress Tracking: System to provide progress on the water usage, Panchayat Rank, Level on quests.
- Badge Unlocking and Sharing: Share your achievement of new badges, and see which ones that are available to unlock based on which quests are completed. User can share on WhatsApp.

## Style Guidelines:

- Primary color: Vibrant green (#7CFC00), evoking growth and prosperity.
- Background color: Light pastel green (#E0F8E0), provides a calm and natural feel.
- Accent color: Golden yellow (#FFD700), symbolizes rewards and achievements.
- Font Pairing: 'Poppins' (sans-serif) for headlines and 'PT Sans' (sans-serif) for body text.
- Use illustrative icons depicting farming activities, crops, and rewards.
- Card-based layout for quests and rewards, easy navigation on smaller devices.
- Subtle animations for level progression and reward redemption.
